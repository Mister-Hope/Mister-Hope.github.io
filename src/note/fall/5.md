# 秋之思【五】

::: center

浅秋，静守时间的光年。

我在季节的暖光里，

追逐那蝶翼的翕动，

捻一瓣细微的花语，

颤醒了一萝青藤的呢喃。

依着情绪的彩铅，

在心事上抹下了

紫色的薰衣草，

火红的枫叶，

金黄的麦田，

含着馨香宁好，

于安逸岁月里面，

留下了静默无言。

♢

阳光，踮起脚尖，

轻盈的亲吻眼睑，

清幽了风的暖意，

吹散了雨丝连绵。

于是我便把秋的诗篇

叙述在这份安宁间。

那细微的欢喜，

你可听见?

:::
