# 秋之思【一】

::: center

毕竟已是秋了，早晚有了些许凉意，

但正午的阳光还有些强烈，

仿佛夏的余热还没有散去。

八月桂花香馨远，

那丝丝缕缕的香沁人心脾，

那浅浅淡雅能瞬间饱满单薄的心，

让内心喜悦如莲。

燥热了一夏，

时光终于慢了下来，

慢得仿佛能抓住岁月的脚步;

日子终于静了下来，

静的仿佛能听见花开的声音。

喜欢这样安恬的静好，

可以坐下来喝一杯茶，

品一些文字，

让一颗浮躁的心在喧嚣中沉淀，

回眸，凝视，闭眼，

纵有万千思绪，

都已风轻云淡。

:::
