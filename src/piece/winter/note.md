# 冬之语【开篇语】

::: center

曾经的我，

将那么多的心绪

都赋予给了秋。

用——

浓墨晕染的文字

写——

秋的静美，

秋的萧瑟，

秋的多情，

秋的恬淡。

而那些禅风流韵的字迹，

自纤细的笔尖奔泻而出，

无一不是袒露着

秋的清喜，秋的憔悴。

如今我说:

我要留一些情意

给冬的白雪，

冬的寒风，

冬的寂静，

冬的沉醉。

于初冬幽凉的早晨，

听风动寒枝的瑟瑟，

听暖意温窗的静寂，

然后，

只在万物清宁的时候，

等一场梅开，

等一幅优美画卷缓缓展现，

一个人，

静静地，静静地，

诉说着冬之语……

:::
