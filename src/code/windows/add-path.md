以 Windows 10 系统为例，直接在左下角搜索栏中输入“系统变量”

![搜索](./assets/pathSearch.png)

在弹出的框中点击环境变量

![系统变量](./assets/path.png)

在弹出的新窗口中找到 path，选中并点击编辑

![系统变量](./assets/pathDetail.png)

点击添加输入想要添加到 path 的路径或者执行文件名即可

![系统变量](./assets/pathAdd.png)
