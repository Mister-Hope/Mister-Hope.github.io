## 题目一：高台抛射的最远水平距离

一小球从距水平地面高度为 $h$ 的平台边缘以大小为 $v_0$ 的初速度抛出。若忽略空气阻力，重力加速度为 $g$，试求：

1. 为使小球落地点与抛出点的水平距离 $x$ 最大，小球抛出时速度方向与水平面的夹角 $\theta$ 应满足什么条件？
2. 该最大的水平距离 $x_{\max}$ 是多少？

::: details 答案

1. 出手角度满足 $\tan\theta = \frac{v_0}{\sqrt{v_0^2 + 2gh}}$。
2. 最远水平距离 $x_{\max} = \frac{v_0}{g} \sqrt{v_0^2 + 2gh}$。

**思路**：

建立直角坐标系，原点 $O$ 为抛出点，水平向右为 $x$ 轴，竖直向上为 $y$ 轴。

小球的运动轨迹方程为（消去时间 $t$）：

$$y = x \tan\theta - \frac{g x^2}{2 v_0^2 \cos^2\theta}$$

利用三角恒等式 $\frac{1}{\cos^2\theta} = 1 + \tan^2\theta$，轨迹方程变为关于 $\tan\theta$ 的方程：

$$y = x \tan\theta - \frac{g x^2}{2 v_0^2} (1 + \tan^2\theta)$$

小球落在地面上，即 $y = -h$。代入并整理，得到一个关于 $\tan\theta$ 的一元二次方程：

$$\frac{g x^2}{2 v_0^2} \tan^2\theta - x \tan\theta + \left( \frac{g x^2}{2 v_0^2} - h \right) = 0$$

若要存在这样的抛射角 $\theta$ 使小球能到达距离 $x$ 处，则该关于 $\tan\theta$ 的方程必须有实数解，即判别式 $\Delta \ge 0$：

$$\Delta = (-x)^2 - 4 \cdot \frac{g x^2}{2 v_0^2} \cdot \left( \frac{g x^2}{2 v_0^2} - h \right) \ge 0$$

化简不等式（约去 $x^2$，因为 $x>0$）：

$$1 - \frac{2g}{v_0^2} \left( \frac{g x^2}{2 v_0^2} - h \right) \ge 0$$

解得：

$$x^2 \le \frac{v_0^2}{g^2} (v_0^2 + 2gh)$$

$$x_{\max} = \frac{v_0}{g} \sqrt{v_0^2 + 2gh}$$

此时 $\Delta = 0$，方程有唯一解：

$$\tan\theta = \frac{-b}{2a} = \frac{x}{2 \cdot \frac{g x^2}{2 v_0^2}} = \frac{v_0^2}{g x_{\max}}$$

代入 $x_{\max}$ 可得：

$$\tan\theta = \frac{v_0}{\sqrt{v_0^2 + 2gh}}$$

:::

## 题目二：定高越障的最远后撤距离

在水平地面上投掷一物体，初速度大小恒为 $v_0$。前方有一高度为 $h$ 的竖直墙壁（$h < \frac{v_0^2}{2g}$）。为了使物体能够越过该墙壁，投掷点距离墙壁的水平距离 $x$ 最大不能超过多少？此时投掷角 $\theta$ 为多少？

::: details 答案

1. 最远后撤距离 $x_{\max} = \frac{v_0}{g} \sqrt{v_0^2 - 2gh}$。
2. 出手角度满足 $\tan\theta = \frac{v_0}{\sqrt{v_0^2 - 2gh}}$。

**思路**：

此题即求解：当轨迹经过点 $(x, h)$ 时，$x$ 的最大值。

沿用题目一中的轨迹方程，此时令 $y = +h$（注意正负号变化）：

$$h = x \tan\theta - \frac{g x^2}{2 v_0^2} (1 + \tan^2\theta)$$
整理为关于 $\tan\theta$ 的一元二次方程：

$$\frac{g x^2}{2 v_0^2} \tan^2\theta - x \tan\theta + \left( \frac{g x^2}{2 v_0^2} + h \right) = 0$$
同理，利用判别式 $\Delta \ge 0$：

$$x^2 - 4 \cdot \frac{g x^2}{2 v_0^2} \cdot \left( \frac{g x^2}{2 v_0^2} + h \right) \ge 0$$
化简得：

$$1 - \frac{2g}{v_0^2} \left( \frac{g x^2}{2 v_0^2} + h \right) \ge 0$$
解得：

$$x^2 \le \frac{v_0^2}{g^2} (v_0^2 - 2gh)$$

即最远距离：

$$x_{\max} = \frac{v_0}{g} \sqrt{v_0^2 - 2gh}$$
对应的角度同样由 $\Delta=0$ 时的根确定：

$$\tan\theta = \frac{v_0^2}{g x_{\max}} = \frac{v_0}{\sqrt{v_0^2 - 2gh}}$$

:::

## 题目三：斜面顶端抛射的最远射程

倾角为 $\theta$ 的斜面，一物体从斜面顶端以大小为 $v_0$ 的初速度抛出。求该物体在斜面上落点与抛出点的最大距离 $L$ 以及此时初速度与水平方向的夹角 $\alpha$。（注：题目中斜面倾角为已知量 $\theta$，抛射角为待求量 $\alpha$）。

::: details 答案

**答案：**

1. 最远距离 $L_{\max} = \frac{v_0^2}{g(1-\sin\theta)}$。
2. 出手角度（相对于水平面） $\alpha = \frac{\pi}{4} - \frac{\theta}{2}$。
   _（即：抛射方向平分“竖直向上”与“斜面方向”的夹角）_

**思路**：

设抛出点为原点，水平向右为 $x$ 轴。

落点在斜面上，坐标满足 $y = -x \tan\theta$（注意 $y$ 向下为负）。

水平位移 $x = v_0 \cos\alpha \cdot t$，竖直位移 $y = v_0 \sin\alpha \cdot t - \frac{1}{2}g t^2$。

由 $y/x = -\tan\theta$，代入得：

$$\frac{v_0 \sin\alpha \cdot t - \frac{1}{2}g t^2}{v_0 \cos\alpha \cdot t} = -\tan\theta$$

解得飞行时间

$$t = \frac{2v_0 (\sin\alpha + \cos\alpha \tan\theta)}{g} = \frac{2v_0 \sin(\alpha+\theta)}{g \cos\theta}$$

斜面射程

$$L = \frac{x}{\cos\theta} = \frac{v_0 \cos\alpha \cdot t}{\cos\theta}$$

代入 $t$ 并整理：

$$L = \frac{2v_0^2 \sin(\alpha+\theta) \cos\alpha}{g \cos^2\theta}$$

利用积化和差公式 $2\sin A \cos B = \sin(A+B) + \sin(A-B)$：

$$L = \frac{v_0^2}{g \cos^2\theta} [\sin(2\alpha+\theta) + \sin\theta]$$

显然，当 $\sin(2\alpha+\theta) = 1$ 时，$L$ 取最大值。

即 $2\alpha + \theta = 90^\circ \Rightarrow \alpha = 45^\circ - \frac{\theta}{2}$。

最大射程为：
$$L_{\max} = \frac{v_0^2}{g \cos^2\theta} (1 + \sin\theta) = \frac{v_0^2}{g (1-\sin^2\theta)} (1 + \sin\theta) = \frac{v_0^2}{g(1-\sin\theta)}$$

:::

## 题目四：斜面底端抛射的最远射程

倾角为 $\theta$ 的斜面，一物体从斜面底端以大小为 $v_0$ 的初速度向斜面上方抛出。求该物体在斜面上落点与抛出点的最大距离 $L$ 以及此时初速度与水平方向的夹角 $\alpha$。

::: details 答案

1. 最远距离 $L_{\max} = \frac{v_0^2}{g(1+\sin\theta)}$。
2. 出手角度（相对于水平面） $\alpha = \frac{\pi}{4} + \frac{\theta}{2}$。

**思路**：

与题目三类似，只是落点坐标满足 $y = x \tan\theta$（落点在 $x$ 轴上方）。
由 $y/x = \tan\theta$ 解得时间：

$$t = \frac{2v_0 \sin(\alpha-\theta)}{g \cos\theta}$$

射程

$$L = \frac{v_0 \cos\alpha \cdot t}{\cos\theta}$$

代入得：

$$L = \frac{2v_0^2 \sin(\alpha-\theta) \cos\alpha}{g \cos^2\theta}$$

利用积化和差公式：

$$L = \frac{v_0^2}{g \cos^2\theta} [\sin(2\alpha-\theta) - \sin\theta]$$

当 $\sin(2\alpha-\theta) = 1$ 时，$L$ 取最大值。

即 $2\alpha - \theta = 90^\circ \Rightarrow \alpha = 45^\circ + \frac{\theta}{2}$。

最大射程为：

$$L_{\max} = \frac{v_0^2}{g \cos^2\theta} (1 - \sin\theta) = \frac{v_0^2}{g(1+\sin\theta)}$$

:::
